# Car Rental System - Java

## About:
This project demonstrates the application of Object-Oriented Programming .  
It has two parts:
1. A Car Rental System that shows how cars can be rented and returned.
2. A Login Program that asks for username and password with 3 tries.

## How it Runs:
- Open the project in Apache NetBeans.
- To test the Car Rental System:
  - Run `Main.java` in the `carrental` package.
- To test the Login Program:
  - Run `LoginSystem.java` in the `login` package.
